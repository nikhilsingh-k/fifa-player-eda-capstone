# Fifa Player Data Analysis - Capstone Project

Capstone project on FIFA Player Data Analysis using Python and Exploratory Data Analysis (EDA).

## Overview
This project explores FIFA player data to uncover insights related to player performance, market value, wages, and growth potential. The analysis supports data-driven decision-making for football clubs, recruiters, and management teams.

## Objectives
- Analyze player demographics and performance metrics
- Understand the relationship between ratings, wages, and market value
- Identify trends in player potential and recruitment strategies
- Generate business insights for sustainable team development

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

## Dataset
The dataset contains over 18,000 FIFA player records with attributes such as:
- Name
- Age
- Nationality
- Overall Rating
- Potential
- Club
- Value
- Wage
- Preferred Foot
- Position

## Key Insights
- Most players are in the 20–30 age range, indicating peak career years
- Higher-rated players generally receive higher wages
- Younger players show stronger future potential
- Balanced recruitment improves long-term sustainability

## Conclusion
This project highlights how exploratory data analysis can provide actionable insights in sports management, helping organizations optimize recruitment, budgeting, and player development strategies.
